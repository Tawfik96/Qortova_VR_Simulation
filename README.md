# Qortova VR Simulation

A Unity 6.3 VR project demonstrating basic safety and animation scenarios.

## Scenes

1. **Electric Shock Animation** – Shows a simulated electric shock sequence.  
2. **Saw Safety Scenario** – Demonstrates proper safety procedures around a saw.

## Setup
1. Make sure lfs is installed (LFS used to track large files <Assets>)
2. Clone the repository:

```bash
git clone https://github.com/Tawfik96/Qortova_VR_Simulation.git
git lfs install
git lfs pull
```
